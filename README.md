# 🎓 Student Result Management System

> A menu-driven Java console application built to practice Object-Oriented Programming by simulating a real-world student record management system.

---

# 📖 About This Project

Every programmer reaches a point where writing small programs is no longer enough. After learning Java fundamentals and beginning Object-Oriented Programming, I wanted to build something that combined multiple concepts into one complete application instead of solving isolated exercises.

This project was created as that challenge.

The goal wasn't simply to make a program that stores student marks—it was to understand how software should be structured using classes, objects, encapsulation, constructors, and modular design. Throughout development, I focused on separating responsibilities between classes, keeping the code organized, and applying OOP principles rather than placing all logic inside a single file.

Although this is a console-based application, the project follows a structure that can later be expanded with databases, file handling, graphical interfaces, or additional modules.

# 🏗️ Architecture

### Project1

* Entry point of the application.
* Starts the Student Management System.

### Student

Responsible for storing information related to a single student.

Attributes include:

* Roll Number
* Name
* Marks
* Total
* Percentage
* Grade

Responsibilities:

* Calculate Total
* Calculate Percentage
* Calculate Grade
* Display Student Information
* Display Student Report

### StudentManagement

Responsible for managing all students.

Handles:

* Student Registration
* Searching
* Editing
* Deleting
* Statistics
* Ranking
* Reports

# 🚀 How to Run

### Clone the repository

```bash
git clone https://github.com/<your-username>/Student-Result-Management-System.git
```

### Move into the project

```bash
cd Student-Result-Management-System
```

### Compile

```bash
javac src/*.java
```

### Run

```bash
java -cp src Project1
```
